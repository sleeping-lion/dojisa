네이버 블로그에 포스팅을 연동 해주는 플러그인 입니다.

== Description ==

본 플러그인은 자신의 네이버 블로그에서 글쓰기 API 키를 발급받은 후 사용 하실 수 있습니다. 


== Installation ==

플러그인 검색에서 naver blog으로 검색하셔서 설치하시면 됩니다.

== Frequently Asked Questions ==

문의사항은 Hints 워드프레스 블로그  http://hints.co.kr/project/naverblogapi-plugin/ 에 댓글남겨주세요. 

1. 설치했는데 어떻게 사용해야 하나요? 방법을 모르겠습니다.
	* 네이버 블로그에서 글쓰기 API를 발급 받으신 후 플러그인을 설치 하시고 설정화면에서 발급 받으신 글쓰기 API키를 설정 하셔서 
	포스팅 시 우측 Naver Blog Options 에서 원하시는 설정값으로 설정 하시어 포스팅 하시면 됩니다.
